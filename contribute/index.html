How to Contribute

We welcome contributions from developers, security researchers, designers, and translators.

Code Contributions

All source code is hosted on GitHub. Please review our coding standards before submitting a pull request. Ensure that you sign your commits with your PGP key.

Translations

We want our tools to be accessible globally. If you can help translate our software into other languages, please reach out to us.
