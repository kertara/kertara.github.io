Kertara Messenger (v0.1.0-alpha)

Description: A Python-based application using Tor for secure connectivity.

Developers: Yonatan Kristian, Joshafat Abdiel Budhianto.

Technical Overview

Kertara Messenger is designed to provide end-to-end encrypted messaging without exposing metadata to central servers. By routing all traffic through the Tor network, it ensures that your IP address and physical location remain hidden.

The backend is written entirely in Python, utilizing standard cryptographic libraries to establish secure socket connections. It operates on a peer-to-peer architecture, meaning there is no central database storing your conversation history.

Features

  • On-the-fly encryption using AES-256 and RSA-4096.
  • Traffic routed exclusively through Tor hidden services (.onion).
  • No central server dependency; true peer-to-peer connection.
  • Self-destructing messages (optional).

Changelog

  • v0.1.0-alpha: Initial public release. Basic peer-to-peer connection established over Tor.
  • v0.0.9: Internal testing phase. Fixed memory leaks in the socket handler.

Ready to Test?
